# api-gate
一个api网关项目，可以当做api开放平台或外网转内网的转发工具 ，
1、项目基于ruoyi搭建：http://www.ruoyi.vip  
2、基于appKey+appSecret的账号  
3、api支持通配符匹配  
4、账号权限分配，和api使用限制  
5、api后端服务器，加权轮询的负载均衡算法，自动检测服务器可用性
